# Eyegway Svelte

This is a monitoring webapp for the `eyegway` package. With the webapp you can connect to a `Hub` and monitor the data in real-time. The monitoring works with dict-like data (i.e. a series of key-value pairs, where values can be generic metadata, images, pointclouds).

## Installation

```console
yarn install
```

## Launch in Dev Mode

```console
yarn dev
```

## Build & Run

```console
yarn build
```

Preview the app:

```console
yarn preview
```

or serve production build (_Node_ adapter required):

```console
node build
```
